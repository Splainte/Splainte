### 🎬 Ce que je fais

Entre deux montages, je construis des **outils sur-mesure** pour fluidifier le travail d'une équipe vidéo : automatisation des rushes, intégrations Premiere Pro, téléchargement de sources, IA en local. Le tout pensé pour des gens qui montent, pas pour des devs.

### 🚀 Projets publics

| Projet | Description |
| --- | --- |
| [**Robloader**](https://github.com/Splainte/Robloader) `🦀 Rust` | Télécharge des vidéos depuis YouTube, TikTok, Instagram, X et Weibo — en un clic. |
| [**Robloader Extension**](https://github.com/Splainte/Robloader-Extension) `🎞️ Premiere` | Robloader directement dans un panneau Premiere Pro : télécharge **et** importe automatiquement dans le chutier. |
| [**Sauron**](https://github.com/Splainte/Sauron) `👁️ Premiere` | Surveille un dossier projet et le miroir automatiquement dans des chutiers Premiere, chemins relatifs. |
| [**StableWarp**](https://github.com/Splainte/StableWarp) `⚙️ Premiere` | Stabilisation + vitesse intelligente pour Premiere Pro. |
